> [!WARNING]
> You **must** legally own the Call of Duty® game you intend to run with this mod. Cracked/Pirated versions of the games are **NOT** supported.

### This repository contains a client mod for each of the following games:

Call of Duty®: Modern Warfare® 2 (2009)
<a href="https://apps.microsoft.com/detail/9P31HFP7L9WT"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- iw4s (<img height="15px" src="assets/github/microsoft.svg" />) - includes controller support emulator<br/>
Call of Duty®: Modern Warfare® 3 (2011)
<a href="https://apps.microsoft.com/detail/9mwlckrnh7lc"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- iw5 (<img height="15px" src="assets/github/microsoft.svg" />) - works with license bypass and controller support emulator<br/>
Call of Duty®: Black Ops II (2012)
<a href="https://apps.microsoft.com/detail/9NFB5MDCWF47"><img height="15px" src="assets/github/microsoft.svg" /></a>
<a href="https://steampowered.com/app/202970"><img height="15px" src="assets/github/steam.svg" /></a>
\- t6s (<img height="15px" src="assets/github/microsoft.svg" />) / t6-mod (<img height="15px" src="assets/github/steam.svg" />) - works partially<br/>
Call of Duty®: Ghosts (2013)
<a href="https://apps.microsoft.com/detail/9PJH49B6K7DD"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- iw6s (<img height="15px" src="assets/github/microsoft.svg" />) - not implemented yet (but will likely use the same logic as s1s, if you use s1s DLL in Ghosts it will also work anyway)<br/>
Call of Duty®: Advanced Warfare (2014)
<a href="https://apps.microsoft.com/detail/9p9m46g36k6w"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- s1s (<img height="15px" src="assets/github/microsoft.svg" />) - works and has all DLCs unlocked<br/>
Call of Duty®: Black Ops III (2015)
<a href="https://apps.microsoft.com/detail/9P5QVB59FXRR"><img height="15px" src="assets/github/microsoft.svg" /></a>
<a href="https://steampowered.com/app/311210"><img height="15px" src="assets/github/steam.svg" />\*</a>
\- t7s (<img height="15px" src="assets/github/microsoft.svg" />) - works for base game + DLC (for mod support, use [ResurgeGDK](https://github.com/Spet001/ResurgeGDK/))<br/>
Call of Duty®: Infinite Warfare (2016)
<a href="https://apps.microsoft.com/detail/9nblggh52xl6"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- iw7s (<img height="15px" src="assets/github/microsoft.svg" />) - doesn't function (currently only contains stubs to analyze the undocumented Windows API `Windows.Services.Store.Internal.StoreContextServer`)<br/>
Call of Duty®: WWII (2017)
<a href="https://apps.microsoft.com/detail/9P3QS4LX8BM9"><img height="15px" src="assets/github/microsoft.svg" /></a>
\- s2s (<img height="15px" src="assets/github/microsoft.svg" />) - works and has all DLCs unlocked<br/>
Call of Duty®: Modern Warfare® (2019)
<a href="https://apps.microsoft.com/detail/9n5h368k9zfb"><img height="15px" src="assets/github/microsoft.svg" /></a>
<a href="https://eu.shop.battle.net/en-us/product/call-of-duty-modern-warfare"><img height="15px" src="assets/github/battle_net.svg" /></a>
\- iw8s (<img height="15px" src="assets/github/microsoft.svg" />) / iw8-mod (<img height="15px" src="assets/github/battle_net.svg" />) - works with license bypass<br/>
